About This Project

This Student Management System is a web-based application built using Flask, HTML, CSS, and SQLite. It allows users to manage student records efficiently, including adding, editing, deleting, searching, and viewing student details.

The goal of this project is to demonstrate a practical implementation of CRUD operations with user-friendly navigation and a simple UI. It's ideal for learning full-stack development with Python and Flask.

Features:

  • Add, update, delete, and search student records
  • Clean and responsive Bootstrap-based UI
  • SQLite as the database for simplicity
  • Navigation structure and reusable components
